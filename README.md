### How to run

1. Clone the repo
```sh
git clone https://github.com/Preetesh21/DanglingPointers
```
2. Install the required packages 
```sh
pip install -r requirements.txt
```
3. Go to the project directory
```sh
cd DanglingPointers
```
4. Run the web app
```sh
python app.py
```
5. Enjoy!
